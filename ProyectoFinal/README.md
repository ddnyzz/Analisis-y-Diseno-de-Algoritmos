# GeoRutas - Algoritmos de Grafos (Zacatecas)

Plataforma para visualizar y ejecutar algoritmos de grafos sobre un mapa geográfico restringido al área de Zacatecas y Guadalupe.

## 🛠️ Stack Tecnológico

* **Backend:** Node.js, Express.js
* **Base de Datos:** SQLite (con `better-sqlite3`)
* **Frontend:** HTML5, CSS3, JavaScript Vanilla (Modular)
* **Mapas:** Leaflet.js con OpenStreetMap/CartoDB

## 🚀 Instalación y Ejecución Local

**Requisito previo:** Tener instalado Node.js 

**1. Instalar dependencias:**
```bash
npm install

ejecutar:
npm run dev
